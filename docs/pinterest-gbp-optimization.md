## AIO (AI Overview) Optimization

Both Pinterest and GBP content influences how AI systems summarize and recommend businesses.

### Key Strategies:

1. **Structured Data on Website**
   - Already have LocalBusiness, Service, FAQPage schemas
   - Consider adding HowTo schema for educational content

2. **Consistent Entity Information**
   - Same business name, services, and descriptions across all platforms
   - Use exact same credentials: "MSN, APRN, FNP-C"

3. **Question-Answer Format**
   - Blog posts with clear Q&A sections
   - GBP Q&A filled out
   - FAQ schema on key pages

4. **Topical Authority**
   - Create content clusters around core topics (HTMA, hormone health, minerals)
   - Interlink related content
   - Pinterest boards mirror website content categories

5. **Review Signals**
   - Positive reviews with specific service mentions
   - Responses that include service keywords
